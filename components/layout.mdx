### Installation

Run the that command to create a new folder and files for your project:

<CodeGroup>

```bash npm
npx shadcn@latest add http://news-rw.vercel.app/r/layout.json
```

```bash pnpm
pnpm dlx shadcn@latest add http://news-rw.vercel.app/r/layout.json
```

```bash yarn
yarn shadcn@latest add http://news-rw.vercel.app/r/layout.json
```

```bash bun
bunx --bun shadcn@latest add http://news-rw.vercel.app/r/layout.json
```

</CodeGroup>
