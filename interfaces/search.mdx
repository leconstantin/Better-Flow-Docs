## Overview

Better Flow's search system provides lightning-fast access to all your content with intelligent filtering and keyboard shortcuts. Find articles, media files, and resources instantly without interrupting your workflow, using either the search bar or the powerful command palette.

## Key Features

- **Universal Search Bar** - Use the prominent search bar to find articles and media files across your entire newsroom with real-time results and keyword highlighting.

- **Command Palette (Ctrl+K)** - Access the powerful "What do you need?" command palette instantly with Ctrl+K for quick navigation and search functionality without using the mouse.

- **Targeted Search Shortcuts** - Use Ctrl+A for article-specific searches and Ctrl+M for media-only searches, allowing you to filter results by content type for more precise discovery.

- **Smart Keyword Highlighting** - Search terms are highlighted in yellow throughout results, making it easy to identify relevant content and understand why specific articles matched your query.

- **Quick Navigation Menu** - The command palette includes direct navigation options to all major sections: Overview, Editor, Media, Activity, Chat, and Analytics for instant section switching.

- **Profile and Settings Access** - Quickly access your profile, settings, and logout options through the command palette without navigating through multiple menus.

- **Real-Time Search Results** - See instant results as you type with live filtering that updates dynamically, helping you find content faster with fewer keystrokes.

- **Cross-Content Search** - Search simultaneously across articles, titles, descriptions, and media file names to ensure you never miss relevant content regardless of where it's stored.

- **Contextual Results Display** - Search results show article status, publication dates, and read times, giving you complete context to identify the exact content you need quickly.
